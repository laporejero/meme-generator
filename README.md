Meme Generator

A React-based web application that allows users to generate custom memes by combining text input with randomly selected images from an external API. Users can input top and bottom text and generate a new meme image with a single click.

This project was built as part of a Scrimba learning module, where I implemented the core functionality, state management, and API integration.

Features
- Fetches meme templates from the Imgflip API
- Random meme image generation
- Editable top and bottom text inputs
- Real-time UI updates based on user input
- Overlay text rendering on images

Tech Stack
- React (useState, useEffect)
- JavaScript (event handling, async data fetching)
- Imgflip API (meme templates)
- CSS for layout and positioning

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
